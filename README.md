### Стек технологий
- Платформа: Visual Studio 2026 (Консольное приложение .NET Framework)
- Язык программирования: C#
- Фреймворк: WPF (Windows Presentation Foundation)
- Для построения графиков функций - System.Windows.Forms.DataVisualization и WindowsFormsHost
- System.Math - для математических вычислений

### Архитектура приложения
- MainWindow - главное окно для навигации
- Page1, Page2, Page3 - страницы с вычислениями

### Функциональные возможности
- Проверка заполненности полей и корректности данных
- Всплывающие подсказки для всех элементов интерфейса
- Графическое отображение формул и результатов
- Построение графика функции и таблицы на третьей странице
- Возможность  изменения значений в при выводе результатов заблокирована
- Подтверждение выхода из приложения
- Очистка полей
